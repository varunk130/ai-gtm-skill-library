# Contributing

Thank you for your interest in contributing to the AI GTM Skill Library.

## How to Contribute

1. Fork the repository
2. Create a feature branch from master
3. Make your changes
4. Submit a Pull Request

## Skill Format

Each skill lives in skills/<skill-name>/SKILL.md and follows this structure:

- YAML frontmatter with 
ame and description
- Heading with skill name and framework
- When to Use section
- What You'll Need section
- Process section with numbered steps
- Output section with deliverables
- Chain Connections to related skills

## Review Process

All Pull Requests require review and approval from the repository owner before merging.
